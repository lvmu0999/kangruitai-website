## 生成工具选择

### 方案 A：通义万相（推荐）
- 阿里云产品，中文支持好
- 免费额度充足
- 商用授权清晰

### 方案 B：Stable Diffusion
- 本地部署，免费
- 需要 GPU
- 控制更精细

### 方案 C：Midjourney
- 质量最高
- 需要 Discord
- 付费订阅

## 生成步骤

1. 准备提示词（已在上文列出）
2. 调用 AI 图片生成 API
3. 保存为对应文件名
4. 放入 `public/images/` 目录
5. 测试网站显示效果

## 后期处理

1. **尺寸调整**：使用 ImageMagick 或 Photoshop
   ```bash
   convert input.jpg -resize 1920x500^ -gravity center -extent 1920x500 output.jpg
   ```

2. **文字叠加**：使用 Canva 或 Photoshop

3. **压缩优化**：
   ```bash
   convert input.jpg -quality 80 output.jpg
   ```
