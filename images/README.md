## 图片来源建议

1. **公司自有图片** - 最佳选择，真实展示公司形象
2. **专业摄影** - 聘请摄影师拍摄公司基地、厂区
3. **正版图库** - 如 Shutterstock、Getty Images、视觉中国等
4. **AI 生成** - 使用 Midjourney、DALL-E 等生成示意图（需注意商用授权）

## 替换步骤

1. 将图片文件放入 `public/images/` 目录
2. 更新对应组件中的图片路径
3. 刷新浏览器查看效果
